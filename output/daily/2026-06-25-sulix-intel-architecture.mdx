## BLUF

Sulix Intelligence 完成重大架构重构：从 Rust 生成 HTML 的单体管线，
演进为 Rust 认知引擎 → MDX 知识资产 → Astro 前端的三层分离架构。
前端已分离为独立仓库 Intel-Web，实现了引擎与展示的彻底解耦。

## Analysis

### 重构背景

之前的架构中，Rust 管线承担了过多职责：
- 数据抓取与清洗
- LLM 主题分析
- HTML 模板渲染
- CSS 设计与生成
- 双语输出

这种"认知引擎 + CMS + 静态站点生成器"的混合体，导致：

### 关键变化

1. **Rust 不再生成 HTML**：所有输出转为 MDX 知识资产
2. **前端独立仓库**：Intel-Web（Astro + Tailwind + Content Collections）
3. **Git Commit 驱动部署**：Engine → MDX → Git Push → Astro 渲染

### MDX 格式的优势

- YAML frontmatter：Git diff 友好，Astro Content Collection 原生支持
- Markdown 正文：人类可读，机器可解析
- 类型安全：Zod schema 校验 frontmatter 字段
