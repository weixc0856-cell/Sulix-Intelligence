## BLUF

AI Agent 能力进入实用化阶段，企业大规模部署加速，创始人应聚焦垂直应用而非基础模型。

## Analysis

**Impact:** 初创公司应优先将 AI Agent 集成到现有工作流中，并关注科研突破带来的新应用场景。

**Geopolitical:** Google DeepMind 在 Gemini 3.5 Flash 中内置计算机使用能力，支持跨平台 Agent 构建；OpenAI 的 GPT-5 Pro 解决了一个三年未解的免疫学谜题；Samsung 在全球部署 ChatGPT Enterprise 和 Codex，成为 OpenAI 最大企业客户之一。

**Supply Chain:** AI Agent 工具成熟降低了自动化开发门槛，但企业级部署需求将推动 MLOps、安全合规等中间件市场增长。模型能力提升加速垂直行业应用，尤其是生命科学领域。

确认了 AI 从模型竞赛转向应用落地的趋势：Agent 能力成为标配，企业采购从实验性转为规模化。创始人应放弃自研基础模型，转向利用 API 构建差异化解决方案。

## Assumptions

- 🔴 **企业大规模部署AI Agent的障碍（安全、合规、集成）可以快速克服** (证据强度: weak)
- 🔴 **基础模型能力持续提升，且通过API可获取足够竞争力** (证据强度: moderate)
- 🟡 **垂直应用领域不存在巨头快速模仿或开源模型挑战** (证据强度: weak)
- 🟡 **科研突破会持续产生新应用场景，且初创能抓住** (证据强度: weak)
- 🔴 **企业愿意为AI Agent支付足够费用，商业模式成立** (证据强度: moderate)

## Causal Chains

- **Trigger:** 模型能力提升
  **Direct:** Agent 内置工具成熟
  **Chain:** 企业大规模部署 → 应用层门槛下降 → 差异化转向垂直整合与数据壁垒
